This folder contains the python_solve_vroom_cloud.py files that perform the VROOM optimisation, call the correct functions and APIs from OSRM, producing a JSON file of the solution (vroom_solution.json), which is then parsed through json_to_geojson.py into the inview_routes.geojson file to be used in QGIS. As well as a JSON, the parsed_itinerary.txt file with a text-based list is produced for readability, as well as a crude map.html map that visualises the journeys.
